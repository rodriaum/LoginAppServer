# Aplicação de Servidor SQL

Esta é uma aplicação que executa um servidor SQL. O servidor SQL atua como a espinha dorsal do sistema, armazenando e gerenciando dados de uma [aplicação de login](https://github.com/rodriaum/login).

## Autenticação de Usuários

A aplicação é projetada para servir como a base de dados para uma [aplicação de login](https://github.com/rodriaum/login) separada. Esta [aplicação de login](https://github.com/rodriaum/login) utiliza os dados armazenados no servidor SQL para autenticar os usuários, garantindo que apenas usuários autorizados tenham acesso.
